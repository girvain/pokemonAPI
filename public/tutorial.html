Intro

The pokemon Page is a collection of D3 visualisation technology, web API's, AJAX and a CSS/Javascript framework. For details on these individual technologies see corisponding pages on this site.

The D3

The D3 that is used to make this page was taken from d3js.org which is a large collection of d3 examples with information and code. The actual d3 project that is used as a data visualisation for the page is by Rob Schmuecker and can be found here.
